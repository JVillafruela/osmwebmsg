# osmwebmsg - Send messages to OpenStreetMap users                                           


```
USAGE:                                                                                                                                                   
   osmwebmsg.php <OPTIONS>                                                                                                                               
                                                                                                                                                         
   Utility so send messages to OpenStreetMap users                                                                                                       
                                                                                                                                                         
                                                                                                                                                         
OPTIONS:                                                                                                                                                 
   -u <username>, --user <username>               OSM user name                                                                                          
                                                                                                                                                         
   -p <password>, --password <password>           OSM user password                                                                                      
                                                                                                                                                         
   -s <subject>, --subject <subject>              Message subject                                                                                        
                                                                                                                                                         
   -m <filename>, --message <filename>            Message                                                                                                
                                                                                                                                                         
   -r <filename>, --recipients <filename>         Recipients list                                                                                        
                                                                                                                                                         
   -h, --help                                     Display this help screen and exit immediately.                                                         
                                                                                                                                                         
   --no-colors                                    Do not use any colors in output. Useful when piping output to other tools or files.                    
                                                                                                                                                         
   --loglevel <level>                             Minimum level of messages to display. Default is info. Valid levels are: debug, info, notice, success, 
                                                  warning, error, critical, alert, emergency.                                                            
                                                             
```
